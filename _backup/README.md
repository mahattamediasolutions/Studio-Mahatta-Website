# Studio Mahatta Website

Landing page for **Studio Mahatta** — a boutique production house for documentaries, brand films, and long-format IP.

## Structure

```
index.html      # Single-page website
test_index.py   # Test suite
```

## Running the site

Open `index.html` directly in a browser — no build step required.

## Running the tests

Requires Python 3:

```bash
python test_index.py
```
